If you have a complaint about the router, you may email the maintainer of this Exit Router. If complaints are related to a particular service that is being abused, it will be considered to remove that service from this router's exit policy, which would prevent this router from allowing that traffic to exit through this Router. This can only be done on an IP+destination port basis.

You also have the option of blocking this IP address if you so desire.
